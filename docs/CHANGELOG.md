## [v1.5.0] - 2026-01-21

### 🎯 数据库架构重大升级

#### 条-款-项三层结构
- **新增 Paragraph 表（款）**
  - 存储条款下的"款"，可选层级
  - 字段：id, articleId, number, content, order
  - 支持级联删除和排序

- **新增 Item 表（项）**
  - 存储款下的"项"，最小存储单元
  - 字段：id, paragraphId, number, content, order
  - 支持精确引用查询（XX法X章X节X条X款X项）

- **Article 表增强**
  - 新增 `section` 字段，实现章和节的分离存储
  - `content` 字段改为可为 null（有款和项时）
  - 优化表结构和约束

#### 解析逻辑升级
- **五层结构识别**
  - 章：`第X章`
  - 节：`第X节`
  - 条：`第X条`
  - 款：自然段分隔
  - 项：`（一）（二）` 或 `1. 2.` 等格式
